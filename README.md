# AI Text Summarizer

A simple web app that summarizes text using AI. Paste your text and get a concise summary instantly. Built with **Next.js**, **Tailwind CSS**, and integrated with an AI model.

---

## Features
- Paste text and get a short summary
- Handles multi-line input
- Responsive UI
- Copy the generated summary to clipboard
- Lightweight and easy to extend

---

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Node.js API route handling AI requests  
- **AI Integration:** Gemini AI (or your preferred AI API)  
- **State Management:** React `useState`
